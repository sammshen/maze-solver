# maze-solver

Terminal-Based Maybe Maze Solver

-------------------------------------------------------------------
Usage:

A maze is represented as an input file, comprising a rectangular grid of the following characters:

S: Entrance (only one), 
F: Exit (only one), 
X: Wall, 
Space: Open

To solve the maze:
cabal run maze-solver -- [maze input file]
